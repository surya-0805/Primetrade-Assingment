# Primetrade.ai Data Science Assignment

## Objective
Analyze the relationship between Bitcoin market sentiment and trader performance.

## Dataset
- Historical Hyperliquid Trading Data
- Bitcoin Fear & Greed Index

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis
- Data Cleaning
- Data Merging
- Exploratory Data Analysis
- Visualizations
- Statistical Insights

## Key Findings
- Fear had the highest trading activity.
- Greed produced the highest average profit per trade.
- Extreme Greed showed the highest win rate.
- Trading fees increased with trade size.
- Profits were concentrated among a small number of traders.
